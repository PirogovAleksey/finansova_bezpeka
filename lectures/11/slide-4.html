Закон 361-IX (Україна)

📋 Повна назва: «Про запобігання та протидію легалізації доходів...»
🏦 Суб'єкти: банки, страхові, платіжні системи, криптобіржі, нотаріуси
💰 Поріг: 400,000 грн — обов'язкове повідомлення ДСФМУ
⚖️ Санкції: штрафи до 10% річного доходу, відкликання ліцензії

AML-система України

Ukrainian AML Framework

📜 Закон 361-IX (2020): основний AML/CFT закон, імплементація 4-ї та 5-ї Директив ЄС
🏛️ ДСФМУ: центральний орган фінмоніторингу (FIU), підпорядкований Мінфіну
🏦 НБУ: регулятор для банків та платіжних систем, перевірки AML-compliance
🪙 НКЦПФР/НБУ: регулювання VASP (криптобіржі) включені як суб'єкти з 2020 року
⚖️ Санкції: штрафи до 74 млн грн, обмеження діяльності, відкликання ліцензії
🌍 Moneyval: Україна проходить оцінку FATF через комітет Moneyval Ради Європи

AML Technology Trends

Технологічні тренди у протидії відмиванню

🤖
AI/ML моделі
Anomaly detection, network analysis, NLP для adverse media — зменшення false positives на 60%
🔗
Аналітика блокчейну (Blockchain Analytics)
Chainalysis, Elliptic — відстеження криптотранзакцій, кластеризація гаманців
🔄
RegTech API
Cloud-based KYC/AML: Sumsub, Jumio, Onfido — верифікація за секунди через API
🤝
Спільна аналітика (Collaborative Analytics)
Privacy-preserving sharing між банками: виявлення cross-bank мереж без розкриття даних

Глобальний ринок RegTech зросте з $12B (2023) до $45B (2028) — AML є найбільшим сегментом.
